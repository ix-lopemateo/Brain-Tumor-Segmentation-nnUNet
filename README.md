# Brain-Tumor-Segmentation-nnUNet
This repository contains the code, documentation and results of an automatic brain tumor segmentation project using nnU-Net on multimodal MRI data.

## Authors
Alejandro Rodríguez Garrido
Antonio Gutiérrez Alcantud
Irene Xue-Feng Lope Mateo

## Dataset
This project uses the Brain Tumor Segmentation (BraTS 2020) dataset, accessed through
Kaggle. The dataset consists of multimodal 3D magnetic resonance imaging (MRI) volumes,
including T1, T1ce, T2 and FLAIR sequences, along with expert manual segmentation labels.

Dataset source (Kaggle):  
https://www.kaggle.com/datasets/awsaf49/brats2020-training-data
